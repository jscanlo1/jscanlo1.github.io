My Projects

• Exploring how emotion detection can be used to leverage fake news

This was the topic of my master's degree. The project looked at how DeepMoji could be used in the realm of Fake News detection. DeepMoji is a language model trained to associate probabilties for over 64 emojis wit some input text. Given that much fake news today takes place in the digital, online sphere, I had the novel idea of applying deepmoji to this task, as emojis represent a uniquely digital way of capturing emotion. The way in which emojis are used is far more deft than one may orginally think. Think of how a 💀 can be used to descibe laughter (dying laughing), and death (obvious). These are two very unrelated sentiments yet we use the one emoji, along with others, to capture it all!

DeepMoji is an LSTM based architecture. I looked at combining it with the transformer-based, then state-of-the-art, BERT architecture.

• ECDSA Node

A learning project I took on while teaching myself the technolgies behind blockchain. It is a simple website containg a cryptogrpahically secure ledger. It uses Node.js and Express.Js for the backend, and React for the front end. Cryptographic keys allow for the secure transfering of money between accounts. A timestamp along with a serverside transaction record is used to prevent replay attacks. note This is not a distributed blockchain solution, only a tool to teach myself the basics of cryptographic keys, hashes and more.
